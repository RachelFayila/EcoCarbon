#  EcoCarbon

EcoCarbon est un projet de développement durable basé sur les solutions naturelles de stockage du carbone, visant à lutter contre le changement climatique tout en améliorant les conditions de vie des communautés locales en Afrique.

##  Objectifs
- Réduire les émissions de CO₂
- Restaurer les écosystèmes naturels
- Créer des emplois verts pour les jeunes et les femmes
- Renforcer la sécurité alimentaire

- JavaScript (DOM)
